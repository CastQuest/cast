const client = CastQuestClient.connect(

);

## Deployment

Contracts are deployed using Foundry:

```bash
cd packages/contracts
forge script script/Deploy.s.sol:DeployScript \
  --rpc-url $RPC_URL \
  --broadcast \
  --verify
```

## Verification

All contracts are verified on:
- Etherscan (Ethereum)
- Basescan (Base)
- Arbiscan (Arbitrum)

View source code and interact directly on block explorers.
