# Getting Started

Welcome to CASTQUEST V3! This guide will help you get started with the platform.

## What You'll Learn

- Setting up your development environment
- Installing dependencies
- Deploying smart contracts
- Using the SDK
- Running the web application

## Prerequisites

Before you begin, ensure you have:

- **Node.js 18+** - [Download](https://nodejs.org/)
- **pnpm 8+** - Install with `npm install -g pnpm`
- **Foundry** - [Install](https://book.getfoundry.sh/getting-started/installation)
- **Git** - [Download](https://git-scm.com/)

## Step 1: Clone the Repository

```bash
git clone https://github.com/castquest/cast.git
cd cast
```

## Step 2: Install Dependencies

```bash
pnpm install
```

This installs all dependencies for all packages in the monorepo.

## Step 3: Set Up Environment Variables

Copy the example environment file:

```bash
cp .env.example .env
```

Edit `.env` and add your values:

```bash
# RPC URLs
MAINNET_RPC_URL=https://mainnet.infura.io/v3/YOUR_KEY
SEPOLIA_RPC_URL=https://sepolia.infura.io/v3/YOUR_KEY
BASE_RPC_URL=https://mainnet.base.org
ARBITRUM_RPC_URL=https://arb1.arbitrum.io/rpc

# Deployer private key (for testnet only!)
PRIVATE_KEY=0x...

# API Keys
ETHERSCAN_API_KEY=...
OPENAI_API_KEY=...
DISCORD_BOT_TOKEN=...

# Database
DB_HOST=localhost
DB_PORT=5432
DB_NAME=castquest
DB_USER=postgres
DB_PASSWORD=...
```

## Step 4: Deploy Smart Contracts

### Install Foundry Dependencies

```bash
cd packages/contracts
forge install OpenZeppelin/openzeppelin-contracts
```

### Compile Contracts

```bash
forge build
```

### Run Tests

```bash
forge test
```

### Deploy to Testnet

```bash
forge script script/Deploy.s.sol:DeployScript \
  --rpc-url $SEPOLIA_RPC_URL \
  --broadcast
```

Save the deployed contract addresses - you'll need them for the next steps.

## Step 5: Build All Packages

```bash
cd ../..
pnpm build
```

This builds:
- SDK
- Agents
- Indexer
- Web app
- Documentation

## Step 6: Start the Development Environment

### Option A: Individual Services

Start services in separate terminals:

```bash
# Terminal 1: Web app
cd apps/web
pnpm dev

# Terminal 2: Indexer
cd packages/indexer
pnpm start

# Terminal 3: Documentation
cd docs-site
pnpm dev
```

### Option B: Docker Compose

Start all services at once:

```bash
cd infra/docker
docker-compose up -d
```

## Step 7: Access the Applications

- **Web App**: http://localhost:3000
- **Indexer API**: http://localhost:3001
- **Documentation**: http://localhost:3002

## Step 8: Connect Your Wallet

1. Open the web app at http://localhost:3000
2. Click "Connect Wallet"
3. Select your wallet (MetaMask, WalletConnect, etc.)
4. Approve the connection

## Step 9: Mint Your First CAST

Using the SDK:

```typescript
import { CastQuestClient } from "@castquest/sdk";
import { ethers } from "ethers";

const client = CastQuestClient.connectWithSigner(
  process.env.RPC_URL,
  process.env.PRIVATE_KEY,
  {
    cast: "0x...", // Your deployed CAST address
    // ... other addresses
  }
);

const tokenId = await client.cast.mint(
  yourAddress,
  "QmIPFSHash...",
  1000, // 10% royalty
  "ipfs://metadata"
);

console.log(`Minted CAST #${tokenId}`);
```

## Common Issues

### Foundry Not Installed

```bash
curl -L https://foundry.paradigm.xyz | bash
foundryup
```

### pnpm Not Found

```bash
npm install -g pnpm
```

### Port Already in Use

Change ports in package.json scripts:

```json
{
  "dev": "next dev -p 3001"
}
```

### Database Connection Error

Ensure PostgreSQL is running:

```bash
# Start with Docker
docker run -d \
  --name postgres \
  -e POSTGRES_PASSWORD=postgres \
  -p 5432:5432 \
  postgres:15
```
